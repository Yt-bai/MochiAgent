### 🏜️Memory —— 给它“记忆”

**Memory（记忆）** —— 是让 Agent 记住重要信息，不用每次都重新问。

---

## 两种记忆：

### 🍏短期记忆（Short-term Memory）

- 只记住当前对话
- 对话结束就清空  

### 🍏长期记忆（Long-term Memory）

- 跨对话的持久化信息  
- 比如用户偏好、历史行为  
- 永久保存  

---

**Memory 是 Agent 从“工具”变成“伙伴”的关键** —— 有记忆的 Agent 能理解你，提供个性化服务。
