# 📘 什么是 Agentic Design Patterns

**《Agentic Design Patterns: A Hands-On Guide to Building Intelligent Systems》**

这是一本由 Google CTO 办公室高级工程师 Antonio Gulli 编写的技术指南。全书围绕 21 种智能体设计模式 展开，从基础模式到高级模式全部覆盖，适用于构建工业级、生产级 Agent 系统（单智能体、多智能体、记忆、规划、工具使用、安全监控等）。它总结了 AI Agent 系统设计中常见的可复用模式和实践，帮助工程师从架构角度构建更可靠、更灵活、更可维护的自主智能体。

*这21种模式其实是不同板块，设计一个Agent时根据需要将不同板块组合使用。
